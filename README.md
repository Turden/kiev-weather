## Requirements

```
Nodejs >= 5.10 https://nodejs.org/en/
```

```
$ npm install webpack -g
```

## Clone

```
$ git clone git@github.com:Turden/kiev-weather.git
```

## Setup external dependencies

```
$ npm install
```

## Build

```
$ npm run build
```

## Usage

```
$ npm run main
```
Open browser at http://localhost:3000/

## Tests

```
$ npm run test
```
